### 1. 概述

FlagCX (Flag Cross-Chip Communication Library) 是支持异构GPU集群混合训练的跨芯片通信库。本文档基于FlagCX代码库分析,提供不同厂商异构混训的完整指南,重点介绍TCCL(TsingMicro)与MCCL(MetaX)等多厂商混合训练场景。

### 1.1 异构混训核心特性
- 多厂商支持: 同时支持10+种CCL后端(NCCL、IXCCL、CNCL、MCCL、XCCL、DUCCL、HCCL、MUSACCL、RCCL、TCCL)
- C2C算法: Cluster-to-Cluster跨集群通信算法
- 自动拓扑检测: 自动识别异构集群拓扑并优化通信路径
- 统一API: 通过统一的API屏蔽底层硬件差异
- Pipeline优化: 支持流水线优化的异构集合通信
- 灵活部署: 支持单NIC和多NIC环境
### 1.2 支持的异构模式

### 2. 异构混训原理

### 2.1 集群自动检测

FlagCX通过设备厂商信息自动检测异构集群拓扑:

```
<span></span>// File: flagcx/core/cluster.cc
flagcxResult_t flagcxCollectClusterInfos(const flagcxVendor *allData,
                                         flagcxCommunicatorType_t *type,
                                         int *homo_rank, int *homo_root_rank,
                                         int *homo_ranks, int *cluster_id,
                                         int *cluster_inter_rank, int *ncluster,
                                         int rank, int nranks) {
  // 初始化为同构模式
  *type = flagcxCommunicatorHomo;
  *ncluster = 1;

  // 扫描所有rank的厂商信息
  std::map<std::string, int> clusterMap;
  clusterMap[allData[0].internal] = 1;
  int numClusters = 1;

  for (int i = 1; i < nranks; ++i) {
    std::string cls = allData[i].internal;
    auto it = clusterMap.find(cls);
    if (it != clusterMap.end()) {
      it->second = it->second + 1;  // 同集群rank计数
    } else {
      clusterMap[cls] = 1;           // 发现新集群
      numClusters += 1;
    }
  }

  // 如果检测到多个集群,切换到混合模式
  if (clusterMap.size() > 1) {
    *type = flagcxCommunicatorHybrid;
    *ncluster = numClusters;
  }

  return flagcxSuccess;
}

```

**检测逻辑**:

```
<span></span>1. 收集所有rank的设备厂商信息(`allData`)
2. 根据厂商名称分组,形成cluster
3. 计算每个cluster的rank数量和cluster总数
4. 确定通信器类型(Homo或Hybrid)

```

### 2.2 厂商映射

FlagCX维护厂商标识到内部枚举的映射:

```
<span></span>flagcxResult_t flagcxFillClusterVendorInfo(const flagcxVendor *allData,
                                           flagcxComm *comm, int *clusterIdData,
                                           int nranks, int ncluster) {
  comm->clusterVendorMap.resize(ncluster);
  for (int i = 0; i < nranks; i++) {
    std::string vendor = allData[i].internal;
    int cluster = clusterIdData[i];

    if (vendor == "NVIDIA") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_NVIDIA;
    } else if (vendor == "ILUVATAR_COREX") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_ILUVATAR_COREX;
    } else if (vendor == "MLU") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_MLU;
    } else if (vendor == "METAX") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_METAX;
    } else if (vendor == "TSINGMICRO") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_TSINGMICRO;
    } else if (vendor == "MTHREADS") {
      comm->clusterVendorMap[cluster] = FLAGCX_VENDOR_MTHREADS;
    }
  }
  return flagcxSuccess;
}

```

### 2.3 C2C通信算法

C2C (Cluster-to-Cluster) 算法是FlagCX异构混训的核心:

```
<span></span>┌─────────────────────────────────────────────────────────────┐
│                  C2C AllReduce 示例                          │
├─────────────────────────────────────────────────────────────┤
│  Cluster 0 (NCCL)      │  Cluster 1 (MCCL)                  │
│  ┌─────┐  ┌─────┐      │  ┌─────┐  ┌─────┐                 │
│  │GPU0 │  │GPU1 │      │  │GPU0 │  │GPU1 │                 │
│  └──┬──┘  └──┬──┘      │  └──┬──┘  └──┬──┘                 │
│     │        │          │     │        │                    │
│     └────┬───┘          │     └────┬───┘                    │
│          │              │          │                        │
│     [Intra-Cluster]    │     [Intra-Cluster]                │
│      AllReduce (NCCL)  │      AllReduce (MCCL)              │
│          │              │          │                        │
│          └──────────────┼──────────┘                        │
│                         │                                   │
│                  [Inter-Cluster]                            │
│                   Send/Recv (FlagCX)                        │
│                         │                                   │
│          ┌──────────────┼──────────┐                        │
│          │              │          │                        │
│     [Intra-Cluster]    │     [Intra-Cluster]                │
│      Broadcast (NCCL)  │      Broadcast (MCCL)              │
│          │              │          │                        │
│     ┌────┴───┐          │     ┌────┴───┐                   │
│  ┌──┴──┐  ┌──┴──┐      │  ┌──┴──┐  ┌──┴──┐                │
│  │GPU0 │  │GPU1 │      │  │GPU0 │  │GPU1 │                │
│  └─────┘  └─────┘      │  └─────┘  └─────┘                │
└─────────────────────────────────────────────────────────────┘

```

**C2C算法步骤**:

1, **Intra-Cluster Reduce**: 每个集群内使用本地CCL(NCCL/MCCL/TCCL)执行AllReduce 2, **Inter-Cluster Exchange**: 集群间通过FlagCX的Send/Recv交换归约结果 3, **Intra-Cluster Broadcast**: 每个集群内广播最终结果

### 3.6 PyTorch DDP训练

### 训练脚本

```
<span></span>#!/usr/bin/env python3
# train_tsm_mccl_hetero.py
"""TCCL + MCCL异构混训PyTorch示例"""

import os
import torch
import torch.nn as nn
import torch.distributed as dist
from torch.nn.parallel import DistributedDataParallel as DDP

class HeterogeneousTrainer:
    """异构混训训练器"""

    def __init__(self, model, config):
        self.config = config
        self.setup_distributed()

        # 初始化模型
        self.device = torch.device(f'cuda:{self.local_rank}')
        self.model = model.to(self.device)
        self.model = DDP(self.model, device_ids=[self.local_rank])

        # 优化器
        self.optimizer = torch.optim.Adam(self.model.parameters(), lr=config['lr'])
        self.criterion = nn.CrossEntropyLoss()

    def setup_distributed(self):
        """初始化异构分布式环境"""
        # 从环境变量获取配置
        self.rank = int(os.environ['RANK'])
        self.local_rank = int(os.environ['LOCAL_RANK'])
        self.world_size = int(os.environ['WORLD_SIZE'])

        # 初始化进程组(使用FlagCX backend)
        dist.init_process_group(
            backend='flagcx',  # 使用FlagCX作为backend
            init_method='env://',
            world_size=self.world_size,
            rank=self.rank
        )

        # 设置设备
        torch.cuda.set_device(self.local_rank)

        # 打印集群信息
        if self.rank == 0:
            print(f"Initialized heterogeneous training:")
            print(f"  World size: {self.world_size}")
            print(f"  Detected clusters: Automatic by FlagCX")
            print(f"  Using C2C algorithm for cross-cluster communication")

    def train_epoch(self, train_loader, epoch):
        """训练一个epoch"""
        self.model.train()
        total_loss = 0

        for batch_idx, (data, target) in enumerate(train_loader):
            data, target = data.to(self.device), target.to(self.device)

            # 前向传播
            self.optimizer.zero_grad()
            output = self.model(data)
            loss = self.criterion(output, target)

            # 反向传播 (自动触发梯度AllReduce)
            loss.backward()

            # 更新参数
            self.optimizer.step()

            total_loss += loss.item()

            if batch_idx % 10 == 0 and self.rank == 0:
                print(f'Epoch {epoch} [{batch_idx}/{len(train_loader)}] '
                      f'Loss: {loss.item():.4f}')

        # 全局同步loss
        avg_loss = total_loss / len(train_loader)
        loss_tensor = torch.tensor([avg_loss], device=self.device)
        dist.all_reduce(loss_tensor, op=dist.ReduceOp.SUM)
        global_loss = loss_tensor.item() / self.world_size

        return global_loss

    def validate(self, val_loader):
        """验证"""
        self.model.eval()
        total_correct = 0
        total_samples = 0

        with torch.no_grad():
            for data, target in val_loader:
                data, target = data.to(self.device), target.to(self.device)
                output = self.model(data)
                pred = output.argmax(dim=1)
                total_correct += pred.eq(target).sum().item()
                total_samples += target.size(0)

        # 全局统计
        stats = torch.tensor([total_correct, total_samples],
                            dtype=torch.float32, device=self.device)
        dist.all_reduce(stats, op=dist.ReduceOp.SUM)

        global_acc = stats[0].item() / stats[1].item()
        return global_acc

    def cleanup(self):
        """清理资源"""
        dist.destroy_process_group()

# 使用示例
def main():
    """主函数"""
    # 配置
    config = {
        'batch_size': 64,
        'lr': 0.001,
        'epochs': 10
    }

    # 创建模型
    model = nn.Sequential(
        nn.Linear(784, 512),
        nn.ReLU(),
        nn.Dropout(0.2),
        nn.Linear(512, 256),
        nn.ReLU(),
        nn.Linear(256, 10)
    )

    # 准备数据集
    from torchvision import datasets, transforms
    transform = transforms.Compose([
        transforms.ToTensor(),
        transforms.Normalize((0.5,), (0.5,))
    ])
    train_dataset = datasets.MNIST('./data', train=True, download=True,
                                   transform=transform)
    val_dataset = datasets.MNIST('./data', train=False, transform=transform)

    # 初始化训练器
    trainer = HeterogeneousTrainer(model, config)

    # 训练
    for epoch in range(config['epochs']):
        train_loss = trainer.train_epoch(
            torch.utils.data.DataLoader(
                train_dataset,
                batch_size=config['batch_size'],
                shuffle=True
            ),
            epoch
        )

        val_acc = trainer.validate(
            torch.utils.data.DataLoader(
                val_dataset,
                batch_size=config['batch_size']
            )
        )

        if trainer.rank == 0:
            print(f'\nEpoch {epoch}:')
            print(f'  Train Loss: {train_loss:.4f}')
            print(f'  Val Acc: {val_acc:.4f}')

    # 清理
    trainer.cleanup()

if __name__ == '__main__':
    main()

```

### 5. 异构混训优化建议

### 5.1 集群分割优化

FlagCX支持通过`FLAGCX_CLUSTER_SPLIT_LIST`环境变量对集群进行细粒度分割:

```

# Cluster 0: 分割为2个子集群
# Cluster 1: 分割为2个子集群
# Cluster 2: 分割为2个子集群

```

- 大规模混训时降低通信复杂度
- 针对不同集群的GPU数量进行负载均衡
- 优化跨集群通信拓扑

### 5.3 Pipeline优化

对于大模型训练,启用Pipeline优化可以隐藏跨集群通信延迟:

```
<span></span># 启用Pipeline优化

# 调整Pipeline参数

export FLAGCX_PIPELINE_CHUNK_SIZE=1M  # Chunk大小

```

**Pipeline优化原理**:

```
<span></span>时间轴:
───────────────────────────────────────────────────────
Sequential:
  [Intra-Reduce] [Inter-Exchange] [Intra-Broadcast]

Pipelined:
  [Chunk1-Intra] [Chunk2-Intra] [Chunk3-Intra] ...
     [Chunk1-Inter] [Chunk2-Inter] ...
        [Chunk1-Bcast] [Chunk2-Bcast] ...
───────────────────────────────────────────────────────

```

### 6. 性能分析与调试

### 6.1 性能监控

```
<span></span>#!/usr/bin/env python3
# hetero_performance_monitor.py
"""异构混训性能监控工具"""

import torch
import torch.distributed as dist
import time
import json
from collections import defaultdict

class HeteroPerformanceMonitor:
    """异构性能监控器"""

    def __init__(self, log_file='hetero_perf.json'):
        self.log_file = log_file
        self.stats = defaultdict(list)
        self.rank = dist.get_rank()
        self.world_size = dist.get_world_size()

    def monitor_allreduce(self, tensor, name='allreduce'):
        """监控AllReduce操作"""
        size_bytes = tensor.numel() * tensor.element_size()

        torch.cuda.synchronize()
        start = time.time()

        dist.all_reduce(tensor, op=dist.ReduceOp.SUM)

        torch.cuda.synchronize()
        end = time.time()

        time_ms = (end - start) * 1000
        bandwidth = (size_bytes * 2 * (self.world_size - 1) / self.world_size) / (time_ms / 1000) / 1e9

        self.stats[name].append({
            'rank': self.rank,
            'timestamp': time.time(),
            'size_bytes': size_bytes,
            'time_ms': time_ms,
            'bandwidth_gbps': bandwidth
        })

    def report(self):
        """生成性能报告"""
        if self.rank != 0:
            return

        print("\n" + "=" * 80)
        print("Heterogeneous Training Performance Report")
        print("=" * 80)

        for op_name, records in self.stats.items():
            if not records:
                continue

            print(f"\n{op_name.upper()}:")
            print("-" * 80)

            times = [r['time_ms'] for r in records]
            bandwidths = [r['bandwidth_gbps'] for r in records]

            print(f"  Operations: {len(records)}")
            print(f"  Avg Time: {sum(times)/len(times):.2f} ms")
            print(f"  Min Time: {min(times):.2f} ms")
            print(f"  Max Time: {max(times):.2f} ms")
            print(f"  Avg Bandwidth: {sum(bandwidths)/len(bandwidths):.2f} GB/s")
            print(f"  Peak Bandwidth: {max(bandwidths):.2f} GB/s")

        # 保存到文件
        with open(self.log_file, 'w') as f:
            json.dump(dict(self.stats), f, indent=2)

        print(f"\nDetailed logs saved to {self.log_file}")

```

### 6.3 常见问题诊断
