### 1. 概述

FlagCX (Flag Cross-Chip Communication Library) 是一个支持异构GPU集群的跨芯片集合通信库。本文档基于FlagCX代码库分析，提供全面的性能优化建议和实践示例。本文基于: FlagCX v0.7+ 。

### 1.1 核心性能特性
- 自动调优系统 (flagcxTuner): 自动选择最优通信配置，实现1.02×-1.26×性能提升
- 成本模型 (Cost Model): 基于延迟和带宽的通信时间预测
- 多NIC支持: 充分利用多网卡带宽
- Pipeline算法: 针对AllReduce、AllGather、ReduceScatter的流水线优化
- 10+种CCL后端: 支持NCCL、IXCCL、CNCL、MCCL、XCCL、DUCCL、HCCL、MUSACCL、RCCL、TCCL
- 零拷贝RDMA: Device-buffer RDMA用于小消息优化
### 1.2 适用场景
- 大规模分布式训练（PyTorch、PaddlePaddle）
- 异构GPU集群通信
- 跨集群通信（C2C算法）
- 多节点NVLink场景
### 2. 自动调优系统

### 2.1 工作原理

FlagCX的自动调优系统通过以下步骤优化通信性能：

1, **候选配置生成**: 生成多种通信器配置候选

2, **性能剖析**: 运行基准测试收集每个配置的性能数据

3, **最优选择**: 基于性能因子选择最佳配置

### 核心代码剖析

```
<span></span>// flagcx_tuner.cc - 调优器初始化
flagcxResult_t flagcxTunerInit(size_t nRanks, size_t nNodes,
                               flagcxDebugLogger_t logFunction,
                               void **context) {
  struct flagcxTunerContext *ctx = new struct flagcxTunerContext;

  // 生成候选配置
  FLAGCXCHECK(generateCandidate(ctx->configList));
  INFO(FLAGCX_TUNING, "Candidate number: %ld.", ctx->configList.size());

  // 环境变量覆盖支持
  const char *tagEnv = flagcxGetEnv("FLAGCX_USE_COMM_TAG");
  const char *nLoopsEnv = flagcxGetEnv("FLAGCX_TUNER_SEARCH_NLOOPS");

  return flagcxSuccess;
}

// 最优通信器选择
static flagcxResult_t findBestComm(struct flagcxTunerContext *ctx,
                                   const struct TunerCollCategory &cat) {
  int bestCommIdx = -1;
  float minTime = FLT_MAX;

  // 基于性能数据计算最优通信器
  for (const auto &idx : ctx->activeCommList) {
    float duration = ctx->timer.getRecord(rkey, true);
    if (duration < minTime * tunerProfileFactor) {
      minTime = duration;
      bestCommIdx = idx;
    }
  }

  return flagcxSuccess;
}

### 3. 成本模型与算法选择

### 3.1 成本模型原理

FlagCX使用成本模型估算不同通信模式的执行时间，基于延迟和带宽参数进行预测。

### 核心代码

```
<span></span>// cost_model.cc - Send/Recv时间计算
float flagcxAlgoTimeEstimator::getSendRecvTime(float curClusterLat,
                                               float remoteClusterLat,
                                               float bw,
                                               int totalCount,
                                               size_t chunkSize) {
  float lat = std::max(curClusterLat, remoteClusterLat);
  size_t bytes = totalCount * getFlagcxDataTypeSize(datatype);
  int steps = (bytes + chunkSize - 1) / chunkSize;

  float time = 0.0;
  for (int s = 0; s < steps; s++) {
    size_t sendSize = std::min(chunkSize, bytes - sizeSent);
    // 时间 = 延迟 + 数据量/带宽（转换为微秒，带宽单位GB/s）
    time += lat + sendSize / (1000 * bw);
  }

  return time;
}

```

### 3.2 算法选择策略

FlagCX支持两种主要算法模式：

### Pipeline vs Sequential

| 算法类型 | 适用场景 | 优势 | 劣势 |
| --- | --- | --- | --- |
| Pipeline | 大消息、高带宽网络 | 隐藏延迟、充分利用带宽 | 实现复杂、小消息overhead较大 |
| Sequential | 小消息、低延迟优先 | 实现简单、小消息效率高 | 带宽利用率较低 |

import torch
import torch.distributed as dist

class FlagCXAlgoSelector:
    """FlagCX算法动态选择器"""

    # 阈值配置（字节）
    SMALL_MSG_THRESHOLD = 1 * 1024 * 1024      # 1MB
    LARGE_MSG_THRESHOLD = 64 * 1024 * 1024     # 64MB

    @staticmethod
    def select_algorithm(tensor_size_bytes):
        """根据消息大小选择算法"""
        if tensor_size_bytes < FlagCXAlgoSelector.SMALL_MSG_THRESHOLD:
            os.environ['FLAGCX_C2C_ALGO'] = 'RING_SEQUENTIAL'
            return 'RING_SEQUENTIAL'
        elif tensor_size_bytes < FlagCXAlgoSelector.LARGE_MSG_THRESHOLD:
            if 'FLAGCX_C2C_ALGO' in os.environ:
                del os.environ['FLAGCX_C2C_ALGO']
            return 'AUTO'
        else:
            os.environ['FLAGCX_C2C_ALGO'] = 'RING_PIPELINED'
            return 'RING_PIPELINED'

    @staticmethod
    def optimize_for_model(model):
        """为模型优化算法"""
        # 计算模型参数总大小
        total_params = sum(p.numel() * p.element_size() for p in model.parameters())

        algo = FlagCXAlgoSelector.select_algorithm(total_params)

"""FlagCX环境变量配置验证工具"""

import os
import sys

class FlagCXConfigValidator:
    """配置验证器"""

    REQUIRED_VARS = {
        'FLAGCX_DEBUG': ['NONE', 'VERSION', 'WARN', 'INFO', 'ABORT', 'TRACE'],
        'FLAGCX_ENABLE_TOPO_DETECT': ['TRUE', 'FALSE'],
    }

    OPTIONAL_VARS = {
        'FLAGCX_IB_HCA': None,  # 自定义值
        'FLAGCX_SOCKET_IFNAME': None,
        'FLAGCX_C2C_ALGO': ['RING_SEQUENTIAL', 'RING_PIPELINED'],
        'FLAGCX_ENABLE_TUNER': ['0', '1'],
        'FLAGCX_USE_COMM_TAG': None,
    }

    def validate(self):
        """验证配置"""
        print("=" * 60)
        print("FlagCX Configuration Validator")
        print("=" * 60)

        errors = []
        warnings = []

        # 检查必需变量
        for var, valid_values in self.REQUIRED_VARS.items():
            value = os.environ.get(var)
            if value is None:
                warnings.append(f"[WARN] {var} not set, using default")
            elif valid_values and value not in valid_values:
                errors.append(f"[ERROR] {var}={value} invalid. Valid: {valid_values}")
            else:
                print(f"[OK] {var}={value}")

        # 检查可选变量
        for var, valid_values in self.OPTIONAL_VARS.items():
            value = os.environ.get(var)
            if value is not None:
                if valid_values and value not in valid_values:
                    errors.append(f"[ERROR] {var}={value} invalid. Valid: {valid_values}")
                else:
                    print(f"[OK] {var}={value}")

        # 输出警告和错误
        print("\n" + "=" * 60)
        if warnings:
            print("Warnings:")
            for w in warnings:
                print(w)

        if errors:
            print("\nErrors:")
            for e in errors:
                print(e)
            return False

        print("\n✓ Configuration valid!")
        return True

    def suggest_optimizations(self):
        """建议优化配置"""
        print("\n" + "=" * 60)
        print("Optimization Suggestions:")
        print("=" * 60)

        # 检查是否启用调优
        if os.environ.get('FLAGCX_ENABLE_TUNER') != '1':
            print("• Consider enabling auto-tuning: FLAGCX_ENABLE_TUNER=1")

        # 检查拓扑检测
        if os.environ.get('FLAGCX_ENABLE_TOPO_DETECT') != 'TRUE':
            print("• Consider enabling topology detection: FLAGCX_ENABLE_TOPO_DETECT=TRUE")

        # 检查调试级别
        debug = os.environ.get('FLAGCX_DEBUG', 'NONE')
        if debug in ['TRACE', 'INFO']:
            print("• Production: Use FLAGCX_DEBUG=WARN to reduce logging overhead")

        # 检查IB配置
        if 'FLAGCX_IB_HCA' not in os.environ:
            print("• Multi-NIC setup: Configure FLAGCX_IB_HCA for optimal bandwidth")

if __name__ == '__main__':

import torch

def benchmark_multi_nic(nic_configs, message_sizes):
    """
    测试不同NIC配置的性能

    Args:
        nic_configs: NIC配置列表
        message_sizes: 测试的消息大小（字节）
    """
    results = {}

    for nic_config in nic_configs:
        print(f"\n{'='*60}")
        print(f"Testing NIC config: {nic_config}")
        print(f"{'='*60}")

        # 设置NIC配置
        os.environ['FLAGCX_IB_HCA'] = nic_config

        # 重新初始化（仅示例，实际需要重启进程）
        # dist.destroy_process_group()
        # dist.init_process_group(backend='flagcx', init_method='env://')

        config_results = []

        for size in message_sizes:
            tensor = torch.randn(size // 4, device='cuda')

            # 预热
            for _ in range(10):
                dist.all_reduce(tensor)
            torch.cuda.synchronize()

            # 测试
            start = time.time()
            for _ in range(100):
                dist.all_reduce(tensor)
            torch.cuda.synchronize()
            end = time.time()

            time_ms = (end - start) * 1000 / 100
            world_size = dist.get_world_size()
            bandwidth = (size * 2 * (world_size - 1) / world_size) / (time_ms / 1000) / 1e9

            config_results.append({

                'time_ms': time_ms,
                'bandwidth_gbps': bandwidth
            })

            print(f"  Size: {size/(1024**2):6.1f}MB | Time: {time_ms:7.2f}ms | BW: {bandwidth:6.2f}GB/s")

        results[nic_config] = config_results

    print("Performance Comparison")
    print(f"{'='*80}")

    for i, config in enumerate(configs[1:], 1):
        print(f"\n{config} vs {baseline}:")

        config_results = results[config]

        for j, (br, cr) in enumerate(zip(baseline_results, config_results)):
            speedup = cr['bandwidth_gbps'] / br['bandwidth_gbps']

<span></span>import torch
import torch.distributed as dist

class OptimizedAllReduce:
    """优化的AllReduce封装"""

    # 阈值配置
    SMALL_MSG = 256 * 1024        # 256KB
    LARGE_MSG = 64 * 1024 * 1024  # 64MB

    @staticmethod
    def allreduce(tensor, op=dist.ReduceOp.SUM):
        """
        优化的AllReduce操作
        根据消息大小选择不同策略
        """
        size_bytes = tensor.numel() * tensor.element_size()

        if size_bytes < OptimizedAllReduce.SMALL_MSG:
            # 小消息：直接AllReduce
            dist.all_reduce(tensor, op=op)

        elif size_bytes < OptimizedAllReduce.LARGE_MSG:
            # 中等消息：使用默认策略
            dist.all_reduce(tensor, op=op)

        else:
            # 大消息：分块处理以减少内存压力
            chunk_size = 64 * 1024 * 1024  # 64MB chunks
            num_chunks = (size_bytes + chunk_size - 1) // chunk_size

            flat_tensor = tensor.view(-1)
            chunk_elements = (flat_tensor.numel() + num_chunks - 1) // num_chunks

            for i in range(num_chunks):
                start = i * chunk_elements
                end = min((i + 1) * chunk_elements, flat_tensor.numel())
                dist.all_reduce(flat_tensor[start:end], op=op)

    @staticmethod
    def allreduce_coalesced(tensors, op=dist.ReduceOp.SUM):
        """
        合并多个小tensor的AllReduce
        减少通信次数
        """
        # 计算总大小
        total_size = sum(t.numel() * t.element_size() for t in tensors)

        if total_size < OptimizedAllReduce.SMALL_MSG:
            # 小消息：逐个AllReduce（延迟低）
            for t in tensors:
                dist.all_reduce(t, op=op)
        else:
            # 大消息：合并AllReduce（吞吐高）
            # 展平并拼接
            flat_tensors = [t.view(-1) for t in tensors]
            coalesced = torch.cat(flat_tensors)

            # 执行AllReduce
            dist.all_reduce(coalesced, op=op)

            # 拆分回原始tensor
            offset = 0
            for t, flat_t in zip(tensors, flat_tensors):
                length = flat_t.numel()
                t.view(-1).copy_(coalesced[offset:offset+length])
                offset += length

# 使用示例
def train_step_optimized(model, data, target):
    """使用优化AllReduce的训练步骤"""
    output = model(data)
    loss = criterion(output, target)
    loss.backward()

    # 优化的梯度AllReduce
    with torch.no_grad():
        for param in model.parameters():
            if param.grad is not None:
                OptimizedAllReduce.allreduce(param.grad)
                param.grad /= dist.get_world_size()

    optimizer.step()
    optimizer.zero_grad()

### 6.2 AllGather优化

```
<span></span>class OptimizedAllGather:
    """优化的AllGather操作"""

    @staticmethod
    def allgather_with_padding(tensor, world_size):
        """
        带填充的AllGather，处理不均匀分片
        """
        local_size = tensor.size(0)

        # 收集所有rank的大小
        size_tensor = torch.tensor([local_size], dtype=torch.long, device=tensor.device)
        size_list = [torch.zeros_like(size_tensor) for _ in range(world_size)]
        dist.all_gather(size_list, size_tensor)

        max_size = max(s.item() for s in size_list)

        # 填充到最大大小
        if local_size < max_size:
            padding = torch.zeros(
                max_size - local_size, *tensor.shape[1:],
                dtype=tensor.dtype, device=tensor.device
            )
            padded_tensor = torch.cat([tensor, padding], dim=0)
        else:
            padded_tensor = tensor

        # AllGather
        gathered_list = [torch.zeros_like(padded_tensor) for _ in range(world_size)]
        dist.all_gather(gathered_list, padded_tensor)

        # 移除填充
        result = []
        for gathered, size in zip(gathered_list, size_list):
            result.append(gathered[:size.item()])

        return result

    @staticmethod
    def allgather_pipeline(tensor, world_size, pipeline_chunks=4):
        """
        流水线AllGather，适用于大tensor
        """
        chunk_size = (tensor.size(0) + pipeline_chunks - 1) // pipeline_chunks
        results = []

        for i in range(pipeline_chunks):
            start = i * chunk_size
            end = min((i + 1) * chunk_size, tensor.size(0))
            chunk = tensor[start:end]

            # AllGather单个chunk
            gathered = [torch.zeros_like(chunk) for _ in range(world_size)]
            dist.all_gather(gathered, chunk)

            results.append(gathered)

        # 重组结果
        final_result = []
        for rank_idx in range(world_size):
            rank_chunks = [r[rank_idx] for r in results]
            final_result.append(torch.cat(rank_chunks, dim=0))

        return final_result

### 6.3 ReduceScatter优化

```
<span></span>class OptimizedReduceScatter:
    """优化的ReduceScatter操作"""

    @staticmethod
    def reduce_scatter_bucketed(tensors, world_size, bucket_size_mb=25):
        """
        分桶ReduceScatter，平衡延迟和带宽
        """
        bucket_size_bytes = bucket_size_mb * 1024 * 1024

        # 按大小分组
        buckets = []
        current_bucket = []
        current_size = 0

        for t in tensors:
            t_size = t.numel() * t.element_size()
            if current_size + t_size > bucket_size_bytes and current_bucket:
                buckets.append(current_bucket)
                current_bucket = [t]
                current_size = t_size
            else:
                current_bucket.append(t)
                current_size += t_size

        if current_bucket:
            buckets.append(current_bucket)

        # 对每个桶执行ReduceScatter
        results = []
        for bucket in buckets:
            # 拼接bucket中的tensor
            flat_tensors = [t.view(-1) for t in bucket]
            concatenated = torch.cat(flat_tensors)

            # 分片
            local_size = (concatenated.numel() + world_size - 1) // world_size
            output = torch.zeros(local_size, dtype=concatenated.dtype, device=concatenated.device)

            # ReduceScatter
            input_list = list(concatenated.chunk(world_size))
            dist.reduce_scatter(output, input_list)

            results.append(output)

        return results

"""FlagCX性能监控工具"""

import torch
import torch.distributed as dist
import time
from collections import defaultdict
import json

class FlagCXMonitor:
    """FlagCX性能监控器"""

    def __init__(self, log_file='flagcx_perf.json'):
        self.log_file = log_file
        self.stats = defaultdict(list)
        self.enabled = True

    def monitor_allreduce(self, tensor, name='allreduce'):
        """监控AllReduce操作"""
        if not self.enabled:
            dist.all_reduce(tensor)
            return

        size_bytes = tensor.numel() * tensor.element_size()

        # 测量时间
        torch.cuda.synchronize()
        start = time.time()

        dist.all_reduce(tensor)

        torch.cuda.synchronize()
        end = time.time()

        time_ms = (end - start) * 1000
        world_size = dist.get_world_size()
        bandwidth = (size_bytes * 2 * (world_size - 1) / world_size) / (time_ms / 1000) / 1e9

        # 记录统计
        self.stats[name].append({
            'timestamp': time.time(),
            'size_bytes': size_bytes,
            'time_ms': time_ms,
            'bandwidth_gbps': bandwidth
        })

    def monitor_allgather(self, output_tensors, input_tensor, name='allgather'):
        """监控AllGather操作"""
        if not self.enabled:
            dist.all_gather(output_tensors, input_tensor)
            return

        size_bytes = input_tensor.numel() * input_tensor.element_size()

        torch.cuda.synchronize()
        start = time.time()

        dist.all_gather(output_tensors, input_tensor)

        torch.cuda.synchronize()
        end = time.time()

        time_ms = (end - start) * 1000
        world_size = dist.get_world_size()
        bandwidth = (size_bytes * world_size) / (time_ms / 1000) / 1e9

        self.stats[name].append({
            'timestamp': time.time(),
            'size_bytes': size_bytes,
            'time_ms': time_ms,
            'bandwidth_gbps': bandwidth
        })

    def report(self):
        """生成性能报告"""
        if dist.get_rank() != 0:
            return

        print("\n" + "=" * 80)
        print("FlagCX Performance Report")
        print("=" * 80)

        for op_name, records in self.stats.items():
            if not records:
                continue

            print(f"\n{op_name.upper()}:")
            print("-" * 80)

            # 计算统计量
            times = [r['time_ms'] for r in records]
            bandwidths = [r['bandwidth_gbps'] for r in records]

            print(f"  Operations: {len(records)}")
            print(f"  Avg Time: {sum(times)/len(times):.2f} ms")
            print(f"  Min Time: {min(times):.2f} ms")
            print(f"  Max Time: {max(times):.2f} ms")
            print(f"  Avg Bandwidth: {sum(bandwidths)/len(bandwidths):.2f} GB/s")
            print(f"  Peak Bandwidth: {max(bandwidths):.2f} GB/s")

        # 保存到文件
        with open(self.log_file, 'w') as f:
            json.dump(dict(self.stats), f, indent=2)

        print(f"\nDetailed logs saved to {self.log_file}")

# 使用示例

<span></span>#!/usr/bin/env python3
# flagcx_ddp_training.py
"""使用FlagCX的完整PyTorch DDP训练示例"""

import os
import torch
import torch.nn as nn
import torch.distributed as dist
from torch.nn.parallel import DistributedDataParallel as DDP
from torch.utils.data import DataLoader
from torch.utils.data.distributed import DistributedSampler

class FlagCXTrainer:
    """FlagCX优化的训练器"""

    def __init__(self, model, train_dataset, val_dataset, config):
        self.config = config
        self.setup_flagcx()

        # 初始化分布式
        dist.init_process_group(backend='flagcx', init_method='env://')
        self.rank = dist.get_rank()
        self.world_size = dist.get_world_size()

        # 设置设备
        torch.cuda.set_device(self.rank)
        self.device = torch.device('cuda', self.rank)

        # 模型
        self.model = model.to(self.device)
        self.model = DDP(self.model, device_ids=[self.rank])

        # 数据加载器
        train_sampler = DistributedSampler(
            train_dataset, num_replicas=self.world_size, rank=self.rank
        )
        self.train_loader = DataLoader(
            train_dataset, batch_size=config['batch_size'],
            sampler=train_sampler, num_workers=4, pin_memory=True
        )

        val_sampler = DistributedSampler(
            val_dataset, num_replicas=self.world_size, rank=self.rank, shuffle=False
        )
        self.val_loader = DataLoader(
            val_dataset, batch_size=config['batch_size'],
            sampler=val_sampler, num_workers=4, pin_memory=True
        )

        # 优化器
        self.optimizer = torch.optim.Adam(self.model.parameters(), lr=config['lr'])
        self.criterion = nn.CrossEntropyLoss()

        # 监控
        from flagcx_monitor import FlagCXMonitor
        self.monitor = FlagCXMonitor(f'perf_rank{self.rank}.json')

    def setup_flagcx(self):
        """配置FlagCX环境"""
        config = self.config

        # 基础配置
        os.environ['FLAGCX_DEBUG'] = config.get('debug_level', 'WARN')
        os.environ['FLAGCX_DEBUG_SUBSYS'] = config.get('debug_subsys', 'INIT,ENV')

        # 自动调优
        if config.get('enable_tuner', True):
            os.environ['FLAGCX_ENABLE_TUNER'] = '1'
            os.environ['FLAGCX_TUNER_SEARCH_NLOOPS'] = str(config.get('tuner_loops', 10))

        # 拓扑检测
        if config.get('enable_topo_detect', True):
            os.environ['FLAGCX_ENABLE_TOPO_DETECT'] = 'TRUE'

        # 网络配置
        if 'ib_hca' in config:
            os.environ['FLAGCX_IB_HCA'] = config['ib_hca']

        # 算法选择
        if 'c2c_algo' in config:
            os.environ['FLAGCX_C2C_ALGO'] = config['c2c_algo']

        if self.rank == 0:
            print("FlagCX Configuration:")
            for k, v in os.environ.items():
                if k.startswith('FLAGCX_'):
                    print(f"  {k}={v}")

    def train_epoch(self, epoch):
        """训练一个epoch"""
        self.model.train()
        self.train_loader.sampler.set_epoch(epoch)

        total_loss = 0
        for batch_idx, (data, target) in enumerate(self.train_loader):
            data, target = data.to(self.device), target.to(self.device)

            self.optimizer.zero_grad()
            output = self.model(data)
            loss = self.criterion(output, target)
            loss.backward()

            # 监控梯度同步
            if batch_idx % 10 == 0:
                for name, param in self.model.named_parameters():
                    if param.grad is not None:
                        self.monitor.monitor_allreduce(
                            param.grad, name=f'grad_{name}'
                        )

            self.optimizer.step()

            total_loss += loss.item()

            if self.rank == 0 and batch_idx % 100 == 0:
                print(f'Epoch {epoch} [{batch_idx}/{len(self.train_loader)}] Loss: {loss.item():.4f}')

        avg_loss = total_loss / len(self.train_loader)

        # 全局平均loss
        loss_tensor = torch.tensor([avg_loss], device=self.device)
        dist.all_reduce(loss_tensor)
        global_avg_loss = loss_tensor.item() / self.world_size

        return global_avg_loss

    @torch.no_grad()
    def validate(self):
        """验证"""
        self.model.eval()

        total_loss = 0
        correct = 0
        total = 0

        for data, target in self.val_loader:
            data, target = data.to(self.device), target.to(self.device)
            output = self.model(data)
            loss = self.criterion(output, target)

            total_loss += loss.item()
            pred = output.argmax(dim=1)
            correct += pred.eq(target).sum().item()
            total += target.size(0)

        # 全局统计
        stats = torch.tensor([total_loss, correct, total], device=self.device)
        dist.all_reduce(stats)

        global_loss = stats[0].item() / len(self.val_loader) / self.world_size
        global_acc = stats[1].item() / stats[2].item()

        return global_loss, global_acc

    def train(self, num_epochs):
        """完整训练流程"""
        best_acc = 0.0

        for epoch in range(num_epochs):
            train_loss = self.train_epoch(epoch)
            val_loss, val_acc = self.validate()

            if self.rank == 0:
                print(f'\nEpoch {epoch}:')
                print(f'  Train Loss: {train_loss:.4f}')
                print(f'  Val Loss: {val_loss:.4f}')
                print(f'  Val Acc: {val_acc:.4f}')

                if val_acc > best_acc:
                    best_acc = val_acc
                    torch.save(self.model.state_dict(), 'best_model.pth')
                    print(f'  Saved best model with acc {best_acc:.4f}')

        # 生成性能报告
        self.monitor.report()

        dist.destroy_process_group()

# 使用示例
def main():
    """主函数"""
    # 配置
    config = {
        'batch_size': 64,
        'lr': 0.001,
        'enable_tuner': True,
        'tuner_loops': 10,
        'enable_topo_detect': True,
        'debug_level': 'INFO',
        'debug_subsys': 'INIT,ENV',
        'ib_hca': '=mlx5_0:1,mlx5_1:1',  # 根据实际配置
        'c2c_algo': 'RING_PIPELINED',
    }

    model = nn.Sequential(
        nn.Linear(784, 512),
        nn.ReLU(),
        nn.Dropout(0.2),
        nn.Linear(512, 256),
        nn.ReLU(),
        nn.Dropout(0.2),
        nn.Linear(256, 10)
    )

    # 假设已有train_dataset和val_dataset
    from torchvision import datasets, transforms
    transform = transforms.Compose([
        transforms.ToTensor(),
        transforms.Normalize((0.5,), (0.5,))
    ])
    train_dataset = datasets.MNIST('./data', train=True, download=True, transform=transform)
    val_dataset = datasets.MNIST('./data', train=False, transform=transform)

    # 训练
    trainer = FlagCXTrainer(model, train_dataset, val_dataset, config)
    trainer.train(num_epochs=10)

if __name__ == '__main__':
    main()

# launch_training.sh - 启动多节点训练

# 配置
NUM_NODES=4
GPUS_PER_NODE=8
TOTAL_GPUS=$((NUM_NODES * GPUS_PER_NODE))

# 生成hostfile
cat > hostfile << EOF
node1 slots=8
node2 slots=8
node3 slots=8
node4 slots=8
EOF

# 启动训练
mpirun -np ${TOTAL_GPUS} \
    -hostfile hostfile \
    --bind-to none \
    -x MASTER_ADDR=node1 \
    -x MASTER_PORT=29500 \
    -x NCCL_DEBUG=INFO \
    -x FLAGCX_ENABLE_TUNER=1 \
    -x FLAGCX_ENABLE_TOPO_DETECT=TRUE \
    -x FLAGCX_IB_HCA='=mlx5_0:1,mlx5_1:1' \
    -x FLAGCX_C2C_ALGO=RING_PIPELINED \
    python flagcx_ddp_training.py

echo "Training completed!"

```

### 10. 总结与最佳实践

### 10.1 性能优化清单

**自动调优**
- 首次部署启用`FLAGCX_ENABLE_TUNER=1`
- 记录最优`FLAGCX_USE_COMM_TAG`用于生产环境
**网络配置**
- 配置`FLAGCX_IB_HCA`选择高性能网卡
- 多NIC场景使用精确匹配（`=mlx5_0:1,mlx5_1:1`）
- 排除低速接口（`FLAGCX_SOCKET_IFNAME=^lo,docker`）
**算法选择**
- 小消息（<1MB）: Sequential算法
- 大消息（>64MB）: Pipeline算法（`FLAGCX_C2C_ALGO=RING_PIPELINED`）
**拓扑优化**
- 首次部署启用`FLAGCX_ENABLE_TOPO_DETECT=TRUE`
- 保存拓扑信息用于后续快速初始化
**监控与调试**
- 开发阶段: `FLAGCX_DEBUG=INFO`
- 生产环境: `FLAGCX_DEBUG=WARN`
- 性能分析: 使用FlagCXMonitor工具

            'FLAGCX_C2C_ALGO': 'RING_PIPELINED',
        },
        'heterogeneous': {
            'FLAGCX_DEBUG': 'INFO',
            'FLAGCX_ENABLE_TOPO_DETECT': 'TRUE',
            'FLAGCX_C2C_ALGO': 'RING_PIPELINED',
        }
    }

    if scenario not in configs:
